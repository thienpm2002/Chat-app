# Chat-app
Ứng dụng chat thời gian thực với các chức năng cơ bản như nhắn tin, gửi ảnh, thông báo tin nhắn mới.  
Dự án được phát triển với mục tiêu rèn luyện kỹ năng Fullstack (MERN Stack).

## 🚀 Công nghệ sử dụng
### Backend
- Node.js (Express)
- MongoDB (Mongoose)
- Socket.IO
- JWT (Refresh Token)

### Frontend
- React.js
- Redux
- Axios

## ✨ Chức năng chính
- Đăng ký, đăng nhập
- Quản lý trạng thái online/offline
- Cập nhật thông tin cá nhân
- Gửi/nhận tin nhắn văn bản
- Gửi ảnh
- Thông báo khi có tin nhắn mới
- Tìm kiếm người dùng

## 📦 Cách cài đặt
Clone project:
```bash
git clone https://github.com/thienpm2002/Chat-app.git
Cài đặt dependencies cho backend:
cd backend
npm install

Cài đặt dependencies cho frontend:
cd frontend
npm install

Chạy ứng dụng:
# Backend
npm start

# Frontend
npm run dev
